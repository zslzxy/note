/**
 * 银行联行号（忽略）
 */

/**
 * 需要存放到安全域中的数据
 * 建议至少需要传收付方账户类型信息及业务种类信息（如果涉及到业务种类）

	- 付款账户 PyerInfo
         /**
         * 付款方名称
         * 在入金业务时，该值为用户的银行账户名称
         * 在出金业务时，该值为用户在支付机构中的支付账户名称
         */
        private String pyerNm;

        /**
         * 付款方账户号
         * 在入金业务时，该值为用户的银行账户编号，即银行账号
         * 在出金业务时，该值为用户在支付机构中的支付账户编号，易极付内部为userId
         */
        private String pyerAcctId;

        /**
         * 付款方账户类型 (例如：银行借记账户，贷记账户、准贷记账户、支付账户)
         * 在入金业务时，该值为用户的银行账户类型，其值为除开PERSONAL_PAY_ACC和PUBLIC_PAY_ACC外的其他枚举值
         * 在出金业务时，该值为用户在支付机构中的支付账户类型，其值只能为PERSONAL_PAY_ACC或PUBLIC_PAY_ACC
         	默认为（   PUBLIC_PAY_ACC("04", "单位支付账户")   ）
         */
        private AccountTypeEnum pyerAcctType;
    
	- 收款账户  PyeeInfo pyeeInfo;
		    /**
             * 收款方账户类型(例如：银行借记账户，贷记账户、准贷记账户、支付账户)

             * 在出金业务时，该值为用户的银行账户类型，其值为除开PERSONAL_PAY_ACC和PUBLIC_PAY_ACC外的其他枚举值
